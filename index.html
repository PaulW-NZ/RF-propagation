RF Propagation Mapper Help

This application allows you to create a path between two points on the map and view the elevation profile along that path.

How to Use:

  • Click on the map to add a start and end point for the path.
  • Alternatively, you can manually enter the coordinates for the start and end points in the sidebar and click "Update Path".
  • The elevation profile will be displayed in the chart below the map.
  • You can adjust the antenna heights for the start and end points to see how it affects the line of sight.
  • Use the options in the sidebar to configure the units, map style, and other settings.
